KKB Epay Drupal module
======================

A Drupal 7 module that adds support for Epay payments system.

This module is created for e-commerce websites that operate primarily in
Kazakhstan. It gives site owners an option to accept credit card payments
and process them at http://epay.kkb.kz, the largest processing center in
the country.

The module is integrated with Ubercart (stable) and Commerce (work in
progress). Integration with Payment and a number of other modules is planned.
On top of that, 3rd-party developers can use complete API of this module and
add Epay support into their custom e-commerce solutions (extensive
documentation and usage examples are included).


Модуль для Drupal 7 для приёма платежей через cистему Epay.

Модуль предназначен для интернет-магазинов в Казахстане, которые желают
производить приём платежей с банковских карт. Обработка платежей производится
в центре авторизации пластиковых карт http://epay.kkb.kz/.

Интеграция модуля с Ubercart завершена и полнофункциональна.

Интеграция модуля с Drupal Commerce приостановлена на неопределённый срок по
причине сложности получения ключа для тестирования платежей и отсутствия
интереса со стороны владельцев сайтов. Если вы заинтересованы в дальнейшем
развитии модуля, откройте новый тикет:

https://drupal.org/node/add/project-issue/kkb_epay

Модуль предоставляет простой API для организации приёма платежей за любые
сервисы без использования Ubercart или Drupal Commerce, подробную документацию
и примеры интеграции.
